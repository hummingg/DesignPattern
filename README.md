# DesignPattern
27种设计模式的C#实现。
比GOF23种多4种：简单工厂、空对象、过滤器、MVC。

# 理解
- 工厂
  - 简单工厂有魔数，会指定方法的参数，用if判断。
  - 工厂方法增加类以替换魔数。
  - 抽象工厂生产成套对象，一个方法生产一套中的一个。
